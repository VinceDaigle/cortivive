# Video Script Templates

## Format 1: Story (60 seconds)

```
HOOK (0-3s): [Emotional hook from hook-library.md]

PROBLEM (3-15s): "For the longest time, I [describe struggle].
I tried [list things that didn't work]. Nothing worked."

TURNING POINT (15-25s): "Then I learned about [cortisol concept].
It turns out, [science fact that reframes the problem]."

SOLUTION (25-45s): "So I started [specific action].
Here's what changed: [result 1], [result 2], [result 3]."

CTA (45-60s): "If this sounds like you, I put together a free
7-Day Cortisol Reset guide. Link in bio. It's the exact
protocol I used in my first week."
```

## Format 2: Quick Tip (15-30 seconds)

```
HOOK (0-3s): "[Tip hook from hook-library.md]"

TIP (3-20s): "Here's what to do: [Step 1]. [Step 2]. [Step 3].
The reason this works is [brief science]."

CTA (20-30s): "Want the full protocol? Free guide in my bio."
```

## Format 3: Myth Bust (30 seconds)

```
HOOK (0-3s): "[Myth bust hook from hook-library.md]"

THE MYTH (3-10s): "Everyone says [common advice].
But here's what the research actually shows..."

THE TRUTH (10-25s): "When cortisol is elevated, [science fact].
That means [implication]. Instead, try [better approach]."

CTA (25-30s): "More in my free cortisol reset guide. Link in bio."
```

## Format 4: Transformation Update (30-60 seconds)

```
HOOK (0-3s): "Week [X] check-in. Here's where I'm at."

UPDATE (3-30s): "This week I [what you did].
I noticed [change 1] and [change 2].
The hardest part was [honesty about challenges].
But [positive reframe]."

TAKEAWAY (30-45s): "The biggest lesson this week:
[actionable insight]."

CTA (45-60s): "Following along? Grab the free 7-Day
Cortisol Reset in my bio to start your own journey."
```

## Writing Tips

- **Be specific**: "I lost 2 inches" beats "I lost weight"
- **Show vulnerability**: Perfection doesn't connect; struggle does
- **Use numbers**: "3 signs", "5 minutes", "7 days"
- **End with value**: Every video should teach something
- **Keep sentences short**: Write for speaking, not reading
