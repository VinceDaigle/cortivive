# AI Image Generation Prompts

## Base Character Prompt

Use this as your foundation for EVERY image to maintain consistency:

```
A natural-looking woman, approximately 40 years old, with [hair color]
shoulder-length hair, warm brown eyes, light makeup, wearing casual
comfortable clothing. Soft warm lighting, home environment background.
Photorealistic style, relatable and approachable appearance.
Not overly attractive or filtered — authentic and real.
```

## Expression Variations

### Neutral / Talking
```
[Base prompt] + looking directly at camera with a slight knowing smile,
as if talking to a close friend. Warm kitchen or living room background.
```

### Concerned / Frustrated
```
[Base prompt] + slightly furrowed brow, concerned expression, hand near
face, looking thoughtful. Wearing a comfortable hoodie. Soft lighting.
```

### Happy / Excited
```
[Base prompt] + genuine bright smile, eyes crinkled with joy, slightly
leaning forward with enthusiasm. Bright natural window light.
```

### Before (Week 1)
```
[Base prompt] + tired eyes with slight dark circles, slightly puffy face,
wearing an oversized t-shirt, dim indoor lighting, looking at camera
with a mix of determination and exhaustion.
```

### After (Week 4)
```
[Base prompt] + visibly more rested, clearer skin, brighter eyes,
slight healthy glow, wearing a fitted casual top, bright natural
lighting, confident gentle smile.
```

## Setting Backgrounds

### Kitchen
```
Modern but lived-in kitchen, warm wood tones, some vegetables or
healthy food visible on counter, soft pendant lighting, cozy atmosphere.
```

### Living Room
```
Comfortable living room with soft couch, warm throw blanket, houseplant
visible, natural window light, calming neutral color palette.
```

### Bedroom (evening routines)
```
Calm bedroom with warm bedside lamp light, neat but not staged,
cozy atmosphere, muted tones, slightly dim for evening feel.
```

## Important Notes

1. **Save your seed number** — this is how you get the same face each time
2. **Keep the same hair color and style** across all images
3. **Lighting should stay consistent** within each "time of day"
4. **Avoid**: heavy makeup, perfect skin, gym/fitness settings, bright colors
5. **Aim for**: authenticity, warmth, relatability
