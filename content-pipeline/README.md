# CortiVive TikTok Content Pipeline

Complete guide for creating and publishing faceless AI persona content for TikTok.

## Overview

**Persona**: A relatable woman ~40 years old documenting her cortisol belly transformation journey.
**Posting Cadence**: 1-2 videos per day
**CTA**: "Link in bio for the free 7-Day Cortisol Reset"

## Workflow

1. **Write script** → Use templates in `scripts/`
2. **Generate character images** → Use prompts in `prompts/image-prompts.md`
3. **Create talking avatar video** → Clipfly or A2E.ai
4. **Add voice** → TTS Vibes or Countik
5. **Edit & add captions** → CapCut
6. **Post with hashtags** → See `prompts/caption-templates.md`

## Content Formats

| Format | Length | Use Case |
|--------|--------|----------|
| Story | 60s | Personal journey updates, "I've had this belly for 3 years..." |
| Quick Tip | 15-30s | Single actionable tip, high save rate |
| Myth Bust | 30s | "Your trainer is wrong about..." — drives comments |
| Transformation Update | 30-60s | Weekly check-ins, before/after |

## Tool Stack (Free/Cheap)

- **Character**: Picsart Persona (free) or Higgsfield
- **Talking Avatar**: Clipfly (free, no watermark) or A2E.ai
- **Voice**: TTS Vibes or Countik (free)
- **Editing**: CapCut (free)
- **Scripts**: Claude / ChatGPT
