# AI Avatar Video Creation Workflow

## Step-by-Step Process

### Step 1: Generate Character Image (Picsart Persona)

1. Go to Picsart Persona (free tier)
2. Use prompts from `prompts/image-prompts.md`
3. Generate 5-10 variations, pick the most relatable one
4. **Save the seed/settings** — you need the SAME character every time
5. Generate multiple expressions: neutral, smiling, concerned, excited
6. Export at highest resolution available

### Step 2: Create Talking Avatar (Clipfly)

1. Go to Clipfly (free, no watermark)
2. Upload your character image
3. Input your script text OR upload audio
4. Select "natural" talking style
5. Generate the video
6. Download in highest quality

**Alternative**: A2E.ai
- Upload image + audio
- More natural lip sync
- Free tier available

### Step 3: Add Voice (TTS)

If not using live audio recording:

1. Go to TTS Vibes or Countik (both free)
2. Paste your script
3. Select a natural female voice (~35-45 age range)
4. **Always use the same voice** for consistency
5. Adjust speed to conversational pace (not too fast)
6. Download the audio file

### Step 4: Edit in CapCut (Free)

1. Import the talking avatar video
2. Import the audio track (if separate)
3. Sync audio to video
4. Add auto-captions (CapCut has built-in feature)
5. Style captions: bold, white text, dark outline, centered bottom third
6. Add background music (low, calming — CapCut library has free options)
7. Add hook text overlay in first 3 seconds (large, eye-catching)
8. Trim to exact length (15s, 30s, or 60s)
9. Export at 1080x1920 (vertical 9:16)

### Step 5: Post to TikTok

1. Upload video
2. Add caption from `prompts/caption-templates.md`
3. Add hashtags (see content calendar)
4. Set cover image (choose the most engaging frame)
5. Post at optimal time (7-9 AM or 7-10 PM in your target timezone)

## Tips for Quality

- **Lighting in AI images**: Use warm, soft lighting — avoid harsh/studio looks
- **Lip sync**: If Clipfly sync looks off, try A2E.ai instead
- **Audio quality**: Use a noise-free environment if recording real voice
- **Caption timing**: Auto-captions should match speech within 0.5s
- **Hook text**: First 3 seconds determine if people keep watching — make it bold
